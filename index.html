Quiz App

Thoughts

My goal which is mostly implemented in the main repo is to have the page serve as a trivia app that can dynamically load from a number of quizzes. I have created questions from the owl and planets quizzes here that will be used to create quizzes that the user can select from on the home page.

Since there is no meaningful page interaction or navigation in the wireframes, I have provided links to the various views
